# Data and Code Availability

The reproduction package for this study is prepared for deposition in a public archive. A private reviewer link should be provided during submission, and the archived record should be made public upon acceptance with a DOI. The package contains scenario definitions and seeds, planner configuration files, episode-level raw simulation outputs, processed result tables, figure source data, latency samples, statistical analysis outputs, and scripts needed to regenerate the reported tables and figures. The source code includes the simulator, RAWA-RHP implementation, implemented baseline planners, analysis scripts, plotting scripts and environment specification. A verification script is provided to reproduce the main paired baseline differences, ablation summaries and latency statistics reported in the manuscript.
