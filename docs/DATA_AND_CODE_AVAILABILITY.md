# Data and Code Availability

The reproduction package for this study is available at
  https://github.com/lhaochen419-del/rawa-rhp-reproduction and archived at
  https://doi.org/10.5281/zenodo.21200028. The package contains scenario
  definitions and seeds, planner configuration files, episode-level raw
  simulation outputs, processed result tables, figure source data, latency
  samples, statistical analysis outputs, and scripts needed to regenerate the
  reported tables and figures. The source code includes the simulator, RAWA-RHP
  implementation, implemented baseline planners, analysis scripts, plotting
  scripts and environment specification. A verification script is provided to
  reproduce the main paired baseline differences, ablation summaries and
  latency statistics reported in the manuscript.
